## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

Twitch Account Authentication Token:

`OATH_TOKEN`

DECAPI API Token:

`DECAPI_TOKEN`

Replicate API Token:

`REP_AUTH`

Discord Invite Link:

`DISCORD`

Channels Bot Runs On:

`SELECTED_CHANNELS`
